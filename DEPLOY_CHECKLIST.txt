================================================================================
STEP 3: Add Environment Variables to Vercel (2 min)
================================================================================

Go to: https://vercel.com/dashboard
Select your Nabda project
Go to: Settings → Environment Variables

Add these 8 variables (copy-paste values from your .env file):

1. NABDA_API_TOKEN
2. NABDA_API_URL
3. NABDA_INSTANCE_ID
4. SUPABASE_URL
5. SUPABASE_SERVICE_ROLE_KEY
6. VITE_SUPABASE_URL
7. VITE_SUPABASE_ANON_KEY
8. GEMINI_API_KEY (optional)
9. VITE_GEMINI_API_KEY (optional)

Click "Save" after each variable.

Expected: All variables saved in Vercel dashboard

================================================================================
STEP 4: Configure Nabda Webhook (1 min)
================================================================================

Go to: https://dashboard.nabdaotp.com
Login with your Nabda account

Click: Webhook Configuration
Toggle: Enable Webhook (ON)

Webhook URL: https://YOUR_VERCEL_DOMAIN/api/webhooks/nabda

Replace YOUR_VERCEL_DOMAIN with your actual Vercel domain
Example: https://nabda-whatsapp.vercel.app/api/webhooks/nabda

Check these boxes:
  ☑ message.sent
  ☑ message.received
  ☑ message.ack

Click: Save Webhook Settings

Expected: Webhook configured and active

================================================================================
STEP 5: Test Deployment (1 min)
================================================================================

Option A: Test health endpoint
  curl https://YOUR_VERCEL_DOMAIN/api/health

  Expected response:
  { "status": "ok", "nabdaConfigured": true, "supabaseConfigured": true }

Option B: Test in your app
  1. Go to: https://YOUR_VERCEL_DOMAIN
  2. Go to Templates tab
  3. Create template: "مرحباً {{name}}!"
  4. Go to Campaigns tab
  5. Select template
  6. Set limit to "1 (Test)"
  7. Click "Start Campaign"

  Expected: Message shows "sent" in logs

================================================================================
🚨 IF SOMETHING GOES WRONG
================================================================================

Error: API key invalid
Fix: Verify NABDA_API_TOKEN in Vercel env vars

Error: Supabase connection failed
Fix: Check SUPABASE_URL and SUPABASE_SERVICE_ROLE_KEY

Error: No contacts found
Fix: Verify contacts in database have ready_to_send=true

Error: Webhook not receiving
Fix: Verify webhook URL ends with /api/webhooks/nabda

Error: Build fails
Fix: Run: npm install && npm run build

Check logs:
  - Vercel: https://vercel.com/dashboard → Deployments → Logs
  - Supabase: https://supabase.com/dashboard → Logs
